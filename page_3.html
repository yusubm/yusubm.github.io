文章推荐: 半导体行业新起之秀 RISC-V

RISC-V 指令集作为半导体行业的新起之秀，备受我国半导体厂商的喜爱，今天来和大家一起了解一下这位“潜力股”到底是何方神圣。

什么是 RISC-V？

RISC-V（Reduced Instruction Set Computing-five）是一个基于精简指令集（Reduced Instruction Set Computing，RISC）原则的开源指令集架构（ISA）。 指令集是软件和硬件之间的接口，可以视作软件与硬件之间沟通的“翻译官”，是芯片产业中不可或缺的基础部分......
